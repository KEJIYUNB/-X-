# 🔧 微信模块替换工具 v9.9.9
> 新人自用脚本 | 一键替换模块 | 自动修权限 | 自动停止微信 |

## 🌟 功能说明
- 一键替换微信模块文件
- 自动删除旧文件并复制新文件
- 自动修复 `/data` 和 `/storage` 权限
- 自动强行停止微信进程
- 支持主应用版 / 分身版
- 支持一键删除所有已移动配置文件

## ✅ 兼容说明
- **脚本类型**：`sh`
- **运行环境**：安卓终端 / MT / 可执行 shell 环境
- **适配场景**：
  - 主应用微信
  - 分身微信
- **当前脚本版本**：`v9.9.9`

## 📦 模块内容
脚本会处理以下目录：

- `X`
- `WechatXposed`
- `x7_3.0`
- `databases`
